# Reviews — заглушки для MVP

> Решение 2026-04-17: на MVP используем **заглушечные отзывы**. После запуска сайта — замещаем реальными (собираем через скриншоты Instagram, Google Reviews бизнеса, запрос у клиентов).
>
> Заглушки обозначены как **анонимизированные образцы** — не выдуманные «Анна К.» без оснований, а реальные паттерны отзывов из конкурирующих сайтов + Instagram @moreminsk.by, переписанные под наш формат. Павел сможет заменить реальными после согласия клиентов.

## Визуальное оформление (по UX-паттерну)

```
┌───────────────────────────────────┐
│  [◯] Анна К.           ★★★★★      │
│       18 июля 2025                │
├───────────────────────────────────┤
│ « Отмечали годовщину свадьбы на   │
│   EVA — лучшее решение за         │
│   последние несколько лет...»     │
│                                   │
│  [Мероприятие: Свадьба]           │
│  [Яхта: EVA]                      │
│                                   │
│  [Источник: Instagram 🔗]         │
└───────────────────────────────────┘
```

- `[◯]` — placeholder avatar (инициалы на цветном кружке — `A` на navy-фоне)
- Кавычки `« »` через `<Accent italic={false}>`
- Теги — compact capsule badges
- Источник — опционально, если есть `sourceUrl`

## Правила замены на реальные

### При сборе реальных отзывов соблюдаем:

1. **Согласие клиента** — явное разрешение на публикацию имени + фото (если есть). Формат: «Ок, можно опубликовать в Telegram» — достаточно.
2. **Анонимизация фамилии** — только первая буква («Анна К.»). Имена без фамилии — OK.
3. **Реальная дата** — дата мероприятия, не дата отзыва.
4. **Ссылка на источник** — если отзыв пришёл из Instagram, добавляем `sourceUrl` с ссылкой на пост.
5. **Минимум 3 предложения** — однострочные отзывы выглядят подозрительно, лучше вообще не публиковать.

### Критерии того, что отзыв «хороший»:

- Конкретика: дата, яхта, повод, имя капитана, детали маршрута
- Эмоция без штампов: не «всё супер» и не «незабываемые впечатления», а что **конкретно** понравилось
- Тон живой, разговорный, не отполированный

### НЕ делаем:

- Не генерируем AI-отзывы под видом реальных
- Не заказываем проплаченные на биржах (Google может забанить в поиске)
- Не используем «5 звёзд» по умолчанию у всех — допустимо 4★ с конструктивной критикой («было бы круто если бы...»)

## Roadmap замены заглушек

| Фаза | Действие | Срок |
|---|---|---|
| **MVP** | Используем 5 заглушек выше | На старте |
| **+1 неделя** | Павел собирает 10 реальных отзывов из Telegram/Instagram/Google Maps с согласием клиентов | После запуска |
| **+1 месяц** | Заменяем заглушки. У каждого отзыва — avatar + sourceUrl | Полный апдейт |
| **+3 месяца** | Автоматизация: форма «Оставить отзыв» на сайте → админ модерирует → публикует | Если формат идёт |
