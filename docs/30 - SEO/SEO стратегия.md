# SEO стратегия

## Принципы

1. **Static-first.** Next.js static export → HTML готов на момент запроса. Идеально для SEO: быстрая отдача, полный контент для краулера без JS-рендеринга.
2. **Страница = запрос.** Один кластер запросов = одна посадочная страница. Не пытаемся запихнуть всё на главную.
3. **Длинный хвост важнее ВЧ.** «Свадьба на яхте Минск недорого» конвертит лучше, чем «аренда яхты».
4. **Локальное SEO обязательно.** Yandex Webmaster + Google My Business + отзывы на Яндекс.Картах.
5. **E-E-A-T через реальный контент.** Реальные отзывы, фото с датами, имена капитанов, юр. адрес.

## План действий по приоритету Pavel Tsarenok

### Фаза 1 — SEO-фундамент (от заказчика)
1. ✅ Переписать `<title>` на всех страницах — см. [[Meta-стратегия]]
2. ✅ Переписать `<meta description>` на всех страницах
3. ✅ Удалить `<meta keywords>` (устарел, игнорируется поисковиками с 2009)
4. ✅ Дописать тексты на существующие страницы услуг
5. ✅ Расширить список страниц услуг — см. [[Карта страниц]]

### Фаза 2 — Структура страницы услуги
```
1. H1 + лид-абзац (3–4 предложения, с ключом)
2. Цена + «Забронировать» (fold)
3. Мини-блок фото
4. Таблица цен (разные пакеты)
5. Галерея событий (mini)
6. Повторная CTA + форма
7. Отзывы (3–5)
8. FAQ (4–6 вопросов — с JSON-LD FAQPage)
9. Футер: контакты, карта, юр.инфо
```

### Фаза 3 — Техническое SEO
- [ ] `sitemap.xml` — генерируется Next.js `sitemap.ts`
- [ ] `robots.txt` — `robots.ts`
- [ ] Schema.org — см. [[Schema.org разметка]]
- [ ] Open Graph + Twitter Cards
- [ ] Hreflang (если будет EN)
- [ ] Canonical URLs
- [ ] `og:image` уникальный на каждую страницу
- [ ] Breadcrumbs (UI + разметка BreadcrumbList)
- [ ] Изображения: `next/image` с LCP-приоритетом, alt на каждую
- [ ] Core Web Vitals: LCP ≤ 2.5s, CLS < 0.1

### Фаза 4 — Внешнее SEO
- [ ] Яндекс.Вебмастер — верификация
- [ ] Google Search Console — верификация
- [ ] Яндекс.Бизнес + Google My Business — фирменные карточки
- [ ] Отзывы на Яндекс.Картах, Google Maps, TripAdvisor
- [ ] Закупка ссылок с локальных каталогов (осторожно, только тематика)
- [ ] Соц. сигналы: Instagram, TikTok, ВК

## Метрики

- Позиции по списку ключей — еженедельно
- Органический трафик — ежемесячно
- Страницы в индексе — еженедельно
- Click-through rate из выдачи — через Search Console
- Заявки с органики — через UTM + Метрику
