## Перелинковка (внутренняя)

### Правила
1. На главной — блок «Услуги» (4–6 самых маржинальных) → ссылки на страницы услуг
2. На главной — блок «Флот» → ссылки на 4 яхты
3. На карточке яхты — «Подходит для: {Свадьба, Девичник, День рождения}» → ссылки на услуги
4. На странице услуги — «Подходящие яхты: {EVA, ALFA...}» → ссылки на флот
5. В футере — полный список услуг + флот
6. Breadcrumbs на каждой странице кроме главной

### Кольца (hub pages)
- **/services** — хаб всех услуг, карточки + описание
- **/fleet** — хаб всего флота, таблица с фильтрами
- **/ceny** — сводный прайс со всеми яхтами и пакетами

## Canonical

- Каждая страница — self-canonical
- Параметры UTM — игнорируются (robots meta noindex для param-версий не нужен при static export)
- Пагинация `/blog` (post-MVP) — `rel="prev/next"`
