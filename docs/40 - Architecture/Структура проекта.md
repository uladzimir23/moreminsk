## Правила импорта (FSD-lite)

```
app → widgets → features → entities → shared
```

- Верхний слой может импортить нижний.
- Между «ветками» одного слоя — через shared.
- Widget не импортит другой widget напрямую (только через shared/ui или features).

## Именование файлов

- Компоненты: `PascalCase.tsx`
- Стили к компоненту: `PascalCase.module.scss` (рядом с компонентом)
- Утилиты/хуки: `kebab-case.ts` (`useFooBar.ts` для хуков — camelCase)
- Типы: `types.ts` (внутри `model/`)
- Page-файлы: Next.js-конвенция (`page.tsx`, `layout.tsx`, `loading.tsx`, `not-found.tsx`)

## Алиасы (tsconfig paths)

```json
{
  "@/app/*": ["./src/app/*"],
  "@/widgets/*": ["./src/widgets/*"],
  "@/features/*": ["./src/features/*"],
  "@/entities/*": ["./src/entities/*"],
  "@/shared/*": ["./src/shared/*"]
}
```
