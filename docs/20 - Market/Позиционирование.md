# Позиционирование

## Кратко (one-liner)

> **Море Minsk — яхты для впечатлений на Минском водохранилище. Прозрачные цены, онлайн-бронирование, видео каждой яхты до брони.**

## Позиционная развилка

| Ось | Варианты | Наш выбор |
|---|---|---|
| Цена | Премиум / Средний / Дешёвый | **Средний** (150–200 BYN/ч — текущий прайс) |
| Целевая | B2C массовый / B2B корпоратив | **B2C массовый** (с допом B2B) |
| Профиль | Узко (только яхты) / Широко (вся вода) | **Узко — яхты + катера** (не размываемся в сап/байдарки) |
| Подача | Эмоция / Рацио | **Эмоция + прозрачность** (фото/видео + чёткая цена) |
| Канал | Сайт-первый / Соц-первый | **Сайт-первый** (SEO + соцсети как поддержка) |

## Конкурентные дифференциаторы (по убыванию силы)

1. **Онлайн-бронирование с календарём** — никто не делает (см. [[Конкурентный анализ#Дыры рынка]])
2. **Видео-обзор каждой яхты** — никто не делает
3. **Прозрачные пакеты с фикс-ценой** — частично у zezet, у остальных «звоните»
4. **Mobile-first UX + скорость** — большинство десктоп-first и тяжёлые
5. **SEO-страницы под длинный хвост** — есть только у yachtminsk
6. **Прозрачный FAQ про практику** (что надеть, еда, дети, погода) — есть у Riviera, но скудно

## Что НЕ обещаем

- Не самый дешёвый (не цена)
- Не самый большой флот (не масштаб)
- Не образовательный центр (не сертификации)
- Не загородный комплекс (не баня + сап + велосипеды)

**Фокус: впечатление + удобство покупки.**

## Tone of Voice

- **Простой**: короткие предложения, минимум канцелярита
- **Тёплый**: «вы», но без излишней фамильярности
- **Конкретный**: цены в цифрах, время в часах, вместимость в людях
- **Без пафоса**: «не самая большая яхта в РБ», «не премиум-сервис», а «комфортная яхта на 8 человек, 200 BYN/час, доступна в субботу»

**Запрещено:**
- «Незабываемые впечатления, которые останутся на всю жизнь...» (штамп)
- «Команда профессионалов с многолетним опытом...» (штамп)
- «Доступные цены» — называй цену
- «Индивидуальный подход» — опиши, в чём он

**Эталон тона:**
> «Яхта BRAVO, до 8 человек. 200 BYN/час, минимум 2 часа. Капитан включён, топливо включено. Ждановичи, выезд из Минска — 25 минут. Свободна в эту субботу с 16:00.»

## Brand promise

**Три обещания, которые сайт должен выполнить:**

1. **Цена за час — на главном экране.** Без «звоните уточнить».
2. **Все фото — реальные.** Никаких стоков. Дата съёмки указана.
3. **Ответ в Telegram — за 30 минут.** Если дольше — извинение и купон.
