# Целевая аудитория

## Сегменты (по поводу аренды)

### 1. Романтика / Свидание (M+F, 25–40)
**JTBD:** «Хочу удивить партнёра/партнёршу, чтобы запомнилось».
- Триггер: годовщина, предложение руки, помолвка, день рождения близкого человека
- Чек: 300–600 BYN (2–3 часа на 2 чел)
- Решающие факторы: атмосфера, фото-зона, шампанское, закат
- Канал: Instagram, поиск «романтическое свидание Минск»

### 2. Свадьбы (24–35, оба пола)
**JTBD:** «Нестандартное место для регистрации/фотосессии/банкета».
- Триггер: подготовка к свадьбе (3–6 мес до)
- Чек: 1500–4000 BYN (полдня + декор + кейтеринг)
- Решающие факторы: вместимость, фото-портфолио свадеб, декор-партнёры
- Канал: свадебные блоги, Instagram, поиск «свадьба на яхте Минск»

### 3. Девичники / Мальчишники (22–32)
**JTBD:** «Тусовка с подругами/друзьями + красивые сторис».
- Триггер: за 1–3 недели до свадьбы подруги/друга
- Чек: 600–1500 BYN (вечер на 6–10 чел)
- Решающие факторы: фото-контент, цена за персону, музыка, бар
- Канал: Instagram, TikTok, рекомендации

### 4. Дни рождения (любой возраст, чаще 25–45)
**JTBD:** «Необычный день рождения для друзей».
- Триггер: 1–2 недели до даты
- Чек: 600–2000 BYN
- Решающие факторы: даты доступности, наличие торта, ведущий
- Канал: Instagram, поиск «где отметить день рождения Минск»

### 5. Корпоративы / Тимбилдинг (HR, 28–45, B2B)
**JTBD:** «Сплочение команды + впечатления для отчёта».
- Триггер: квартальные/годовые мероприятия, найм, празднования
- Чек: 3000–10000 BYN (день, несколько яхт, регата)
- Решающие факторы: масштаб, безнал, документы, программа (регата/квест)
- Канал: B2B-сети, рекомендации event-агентств, LinkedIn

### 6. Семьи с детьми (родители 30–45)
**JTBD:** «Активный выходной для семьи».
- Триггер: лето, выходной, каникулы
- Чек: 300–600 BYN (2 часа)
- Решающие факторы: безопасность детей, спасжилеты, длительность
- Канал: поиск «отдых с детьми Минск», семейные блоги

### 7. Туристы (приезжие, 25–55)
**JTBD:** «Что нестандартного посмотреть в Минске».
- Триггер: визит в город (1–3 дня)
- Чек: 200–500 BYN
- Решающие факторы: английский язык, простое бронирование, отзывы на TripAdvisor
- Канал: TripAdvisor, Booking, Google Maps

### 8. Фотографы / Контент-мейкеры (B2B2C)
**JTBD:** «Локация для съёмки клиента».
- Триггер: проект клиента
- Чек: 200–400 BYN (1–2 часа фотосессии)
- Решающие факторы: партнёрские цены, гибкость по времени
- Канал: профессиональные сообщества, рекомендации

### 9. Учащиеся яхтингу (M, 25–50)
**JTBD:** «Хочу научиться/получить права».
- Триггер: интерес к парусному спорту, путешествия
- Чек: 500–2000 BYN курс
- Решающие факторы: сертификат, тренеры, программа
- Канал: поиск «обучение яхтингу Минск», тематические сообщества

## Приоритеты для MVP

🔥 **Топ-3 для запуска** (по объёму трафика и марже):
1. **Романтика / Свидание** — высокая частотность, низкий порог входа
2. **Свадьбы** — высокий чек, длинный цикл, важна репутация
3. **Дни рождения** — массовый сегмент

🟡 **Второй приоритет:** девичники, семьи, корпоративы.

🟢 **Долгосрочно:** туристы (нужен EN), обучение, фотографы (партнёрка).

## Психография

**Что общего у всех сегментов:**
- Принимают решение быстро (в течение 1–7 дней)
- Доверяют визуалу и отзывам больше, чем тексту
- Ожидают ответ в мессенджере < 1 часа
- Раздражает: непонятная цена, скрытые комиссии, заявки без обратной связи

**Что критично на сайте:**
- Цена — на виду, без «*звоните уточнить»
- Фото — много и реальные (не стоки)
- Связь — Telegram/WhatsApp в один клик
- Соц. доказательство — отзывы с фото клиента (не безымянные)
