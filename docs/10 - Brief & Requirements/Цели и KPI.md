# Цели и KPI

## Бизнес-цели

1. **Увеличить органический трафик** из поиска (Yandex/Google) — заказчик явно делает упор на SEO.
2. **Поднять конверсию в заявку** через продающую структуру (лид → прайс → галерея → CTA → отзывы).
3. **Закрыть рыночные дыры** конкурентов — онлайн-бронирование, видео, прозрачные пакеты.

## KPI (предлагаемые — требуют согласования)

### SEO
- Топ-10 в поиске по 30+ запросам кластера «аренда яхты Минск» к 6 мес.
- 15+ продвигаемых страниц услуг в индексе.
- Lighthouse SEO ≥ 95, Performance ≥ 85 на mobile.

### Контент
- 15+ страниц услуг (по образцу yachtminsk + сверху)
- 4 карточки яхт с полным описанием, ценами, галереей
- FAQ ≥ 12 вопросов с разметкой Schema.org/FAQPage
- ≥ 50 фотографий и 4 видео-обхода яхт

### Конверсия
- Форма заявки на каждой странице услуги
- 3-шаговое бронирование (без регистрации)
- Telegram/WhatsApp-кнопка на липкой панели на mobile
- Цель аналитики: заявка / звонок / клик в мессенджер

### Скорость
- LCP ≤ 2.5s (mobile, 4G)
- CLS < 0.1
- Static export → CDN

## Out of scope (для MVP)

- Личный кабинет клиента
- Платежи онлайн (только заявка → менеджер)
- Мультивалютность
- Блог (после MVP)
