# Объём работ MVP

## In scope (MVP — релиз 1.0)

### Страницы
- Главная
- 4 карточки яхт (EVA, ALFA, MARIO, BRAVO)
- 15+ страниц услуг — см. [[../30 - SEO/Карта страниц]]
- Прайс
- Галерея
- Контакты
- Соглашение / Политика конфиденциальности

### Функционал
- Многоязычность RU (+ EN опционально)
- Форма заявки → email + Telegram-уведомление менеджеру
- Click-to-call / click-to-Telegram кнопки
- Sticky CTA на mobile
- Карта Яндекс/Google в контактах
- 3 уровня навигации (хедер, футер, breadcrumbs)

### SEO
- `generateMetadata` на каждой странице
- Schema.org: LocalBusiness, Service, FAQPage, BreadcrumbList, Product (для яхт)
- `sitemap.ts` + `robots.ts`
- Open Graph + Twitter Cards
- Hreflang (если EN)

### Инфраструктура
- Next.js 16 static export → деплой на VPS / Vercel
- Search Console + Yandex Webmaster подключение
- Метрика + GA4
- Бекапы

## Post-MVP (релиз 2.0+)

- Онлайн-бронирование с календарём занятости яхты
- Видео-обходы каждой яхты (60 сек, вертикаль)
- Пакеты «под ключ»: яхта + фотограф + декор + кейтеринг
- Подарочные сертификаты с покупкой онлайн
- Партнёрская программа (фотографы, кейтеринг, ведущие)
- Блог / журнал
- Программа лояльности

## Допущения

- Заказчик предоставит фото/видео контент или мы организуем съёмку (отдельный этап)
- Все тексты согласовываются с заказчиком до публикации
- Бэкенд для формы — простой serverless (Resend / Telegram Bot API), без CMS на старте
- CMS (Sanity/Strapi/PocketBase) — после MVP, если потребуется обновление контента без разработчика
