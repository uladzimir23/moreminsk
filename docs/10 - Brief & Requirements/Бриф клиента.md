> 1. Переписать title на всех страницах
> 2. Переписать description на всех страницах
> 3. Убрать тег keywords

>
> Только после этого заниматься конверсионными моментами: добавить форму, конверсионные блоки.

## Расшифровка приоритетов

### Приоритет 1 — SEO-фундамент
1. Переписать `<title>` на всех страницах
2. Переписать `<meta description>` на всех страницах
3. Удалить устаревший `<meta keywords>`
4. Дописать тексты на существующих страницах услуг
5. Расширить сетку услуг по образцу yachtminsk.com и сверху

### Приоритет 2 — Продающая структура
Каждая страница услуги должна следовать формуле:

```
[Лид-абзац + кнопка + цена + фото]
       ↓
[Таблица цен]
       ↓
[Галерея фотографий с мероприятий]
       ↓
[Кнопка "Заказать"]
       ↓
[Отзывы]
```

После отзывов структура «уже не важна для пользователя» — то есть юзер уже либо ушёл, либо принял решение.

### Приоритет 3 — Конверсия
Только после SEO + продающей структуры:
- Форма обратной связи
- Конверсионные блоки (предположительно: спецпредложения, скидки, акции, всплывающие виджеты)

## Что НЕ просили (но имеет смысл предложить)
- Онлайн-бронирование с календарём занятости — рыночная дыра ([[20 - Market/Конкурентный анализ]])
- Видео-обходы яхт — никто не делает
- Локализация EN/BE — для туристов
- PWA / офлайн-режим — для слабого мобильного интернета у воды
