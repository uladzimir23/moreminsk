# Идеи развития

> Свалка идей без приоритезации. Используются как источник для планирования пост-MVP. Приоритет выставляется при переводе идеи в задачу.

## 🔥 High Impact (кандидаты на 2-й релиз)

### 1. Онлайн-бронирование с календарём занятости
Клиент видит свободные даты в реальном времени, выбирает слот, подтверждает аванс картой.
**Почему важно:** никто из конкурентов не делает. Отличие № 1.
**Что нужно:** админка менеджера (мини-CMS), календарь на сайте, интеграция с платёжкой (ЕРИП / BePaid / Stripe для EN).
**Риск:** перелив в сервер-сайд (Next.js SSR/ISR или headless backend).

### 2. Видео-обход каждой яхты (60 сек)
Вертикальное видео «экскурсия по яхте» — человек видит, куда сядет, где столик, где руль.
**Почему важно:** снимает возражения до звонка, дешёвый production.
**Что нужно:** 1 день съёмки по 4 яхтам, монтаж.

### 3. Пакеты «под ключ»
Фиксированные комплекты:
- «Свадьба-light» = яхта + фотограф + декор + торт — за 2800 BYN
- «Девичник-закат» = яхта + шампанское + фотограф — за 900 BYN
**Почему важно:** упрощает выбор, повышает средний чек, ещё одна дыра рынка.
**Что нужно:** партнёрские соглашения с фотографами, декораторами, кондитерскими.

### 4. Instagram Stories Aggregator
Виджет на главной с последними 10 сторис из @moreminsk.by.
**Почему важно:** живой контент, доверие, не нужно обновлять руками.
**Что нужно:** Instagram Graph API или 3rd-party виджет (SnapWidget, Taggbox).

### 5. Страница «Где именно мы плаваем»
Карта с маршрутами, точками купания, любимыми бухтами.
**Почему важно:** контент под «что делают на яхте» — SEO + снятие возражений.
**Что нужно:** капитан рисует 5–7 маршрутов, дизайнер оформляет карту.

## 🟡 Medium Impact (после high-priority)

### 6. Подарочные сертификаты онлайн
Покупка → сразу PDF на email, срок 6 мес, уникальный код.
**Почему важно:** прибыль до события + виральность (дарят → новый клиент).

### 7. Программа лояльности
Приводишь друга → 10% скидка на следующую бронь. Код в личном профиле (лайт-кабинет).

### 8. Блог / журнал
- «Как подготовиться к свадьбе на яхте: чек-лист за 3 месяца»
- «Топ-10 идей для девичника в Минске 2026»
- «Что не стоит надевать на яхту» (гумор + SEO)
**Почему важно:** длинный хвост + контент-маркетинг в соцсети.

### 9. TripAdvisor / Яндекс.Карты — системное наращивание отзывов
После каждого мероприятия — автосмс клиенту «Оставьте отзыв на Яндекс.Картах, скидка 5% на следующий заказ».
**Почему важно:** локальное SEO и доверие.

### 10. Партнёрская программа для event-агентств
Комиссия 10% агентству за привод клиента.
**Почему важно:** канал продаж B2B через агентства.

### 11. Dog-friendly отдельная страница
`/s-sobakoj` — отдельная лендоса для владельцев собак. У yachtminsk это УТП, но они его плохо раскрывают.
**Почему важно:** нишевой трафик, мало конкурентов.

### 12. Рассылка «Свободные даты на выходные»
Еженедельное письмо/канал-пост: «Осталось 3 слота на этот уикенд — скидка 15%».
**Почему важно:** заполняет пустые окна, повторные клиенты.

## 🟢 Nice-to-have

### 13. AR-preview яхты
QR на сайте → наведи телефон → яхта в твоей комнате в масштабе.
**Почему важно:** вау-эффект. Технически сложно, но 1:1 никто не делает.

### 14. Интеграция с Telegram Mini App
Мини-приложение в Telegram: выбор яхты → календарь → оплата. Без ухода из мессенджера.
**Почему важно:** большая часть аудитории в Telegram; ниже friction.

### 15. Корпоративные абонементы
Годовой абонемент на компанию: 10 выходов по цене 8.
**Почему важно:** B2B с предоплатой, стабильный доход.

### 16. Мерч
Футболки, кепки с логотипом Море Minsk — дарим при бронировании от X.
**Почему важно:** ходящая реклама + сувенир.

### 17. Детский лагерь «Юный яхтсмен»
Лето, 5 дней, дети 10–15. Паруса, безопасность, соревнования.
**Почему важно:** новый сегмент (родители + дети), повторный клиент через 1–2 года.

### 18. Обучение на права (категория «парус», ВВП РБ)
Как у Meridian, но дешевле и с нашим брендом.
**Почему важно:** несезонная услуга (теория — зимой).

### 19. Сотрудничество с HR-tech для корпоративов
Интеграция с корпоративными event-платформами (например «Прямо в офис», HR-бюро).

### 20. Виртуальный тур по пирсу
360° панорама пирса/зоны посадки — люди приезжают и не теряются.
