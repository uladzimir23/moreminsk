# Attachments

Папка для вложений: скриншоты, PDF-брифы, референсы дизайна, фото клиента.

## Конвенция

- `screenshots/` — скриншоты сайта, конкурентов, админок
- `references/` — изображения-референсы (Figma exports, Pinterest)
- `briefs/` — PDF-брифы от заказчика
- `photos/` — RAW/обработанные фото для сайта (только рабочие версии; финальные — в репозитории проекта `public/`)

## Именование файлов

`YYYY-MM-DD_<slug>_<номер>.ext` — например `2026-04-16_brief-tsarenok_01.png`.
